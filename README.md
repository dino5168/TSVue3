# TSVue3 文件紀錄

### 如何上傳檔案到 Github

```sh
git init

git add README.md

git commit -m "first commit"

git branch -M main

git remote add origin https://github.com/dino5168/TSVue3.git

git push -u origin main
```

### 設定本機與 GitHub 的連結。

```sh
//先安裝 git
git config --global user.name "你的GitHub名稱"
git config --global user.email "你的GitHub註冊信箱"

```
